南順宮 PWA 網站

入口檔案：
index.html

發佈方式：
請將整個資料夾內容上傳到支援 HTTPS 的網站空間。PWA 安裝與 Service Worker 離線快取需要 HTTPS 或 localhost 環境才會啟用。

網站內容：
- RWD 響應式設計，支援手機、平板、電腦
- PWA manifest 與 Service Worker
- 首頁、信仰源流、參拜資訊、近期動態、相片集、聯絡交通
- FB 即時同步區塊使用 Facebook 官方 Page Plugin 載入粉絲專頁 timeline
- 手機可直接撥電話、開 Google 地圖、前往 Facebook 粉絲專頁

FB 同步說明：
目前採用官方嵌入方式，不需要粉專管理員權杖。若要把貼文資料存入網站資料庫或自行排版，需要另外申請 Meta Graph API、建立 App、取得粉專管理權限與 Page Access Token，並依 Meta 審核規範使用。
